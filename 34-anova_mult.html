12.7 Test de permutation pour l‚ÄôANOVA √† 2 facteurs de classification

Quand les donn√©es ne r√©pondent pas aux conditions d‚Äôapplications d‚Äôune ANOVA param√©trique √† 2 ou plusieurs facteurs de classification, il est possible, comme alternative √† l‚ÄôANOVA non-param√©trique, de faire des tests de permutation pour calculer les valeurs p.

# Test de permutation pour une ANOVA √† 2 facteurs
# Ter Braak cr√©e des r√©sidus √† partir des moyennes des cellules,
# puis effectue des permutations entre toutes les cellules
# Cela peut √™tre fait en prenant les r√©sidus du mod√®le complet.
# modifi√© depuis le code √©crit par David C. Howell
# http://www.uvm.edu/~dhowell/StatPages/More_Stuff/Permutation%20Anova/PermTestsAnova.html
nreps <- 500
dependent <- Stu2wdat$rdwght
factor1 <- as.factor(Stu2wdat$sex)
factor2 <- as.factor(Stu2wdat$location)
my.dataframe <- data.frame(dependent, factor1, factor2)
my.dataframe.noNA <- my.dataframe[complete.cases(my.dataframe), ]
mod <- lm(dependent ~ factor1 + factor2 + factor1:factor2,
  data = my.dataframe.noNA
)
res <- mod$residuals
TBint <- numeric(nreps)
TB1 <- numeric(nreps)
TB2 <- numeric(nreps)
ANOVA <- summary(aov(mod))
cat(
  " L'ANOVA standard pour ces donn√©es suit ",
  "\n"
)
F1 <- ANOVA[[1]]$"F value"[1]
F2 <- ANOVA[[1]]$"F value"[2]
Finteract <- ANOVA[[1]]$"F value"[3]
print(ANOVA)
cat("\n")
cat("\n")
TBint[1] <- Finteract
for (i in 2:nreps) {
  newdat <- sample(res, length(res), replace = FALSE)
  modb <- summary(aov(newdat ~ factor1 + factor2 +
    factor1:factor2,
  data = my.dataframe.noNA
  ))
  TBint[i] <- modb[[1]]$"F value"[3]
  TB1[i] <- modb[[1]]$"F value"[1]
  TB2[i] <- modb[[1]]$"F value"[2]
}
probInt <- length(TBint[TBint >= Finteract]) / nreps
prob1 <- length(TB1[TB1 >= F1]) / nreps
prob2 <- length(TB2[TB1 >= F2]) / nreps
cat("\n")
cat("\n")
print("R√©√©chantillonnage comme dans ter Braak avec r√©√©chantillone non restraint des cellules r√©siduelles. ")
cat(
  "La probabilit√© pour l'effet d'Interaction est ",
  probInt, "\n"
)
cat(
  "La probabilit√© pour l'effet du Facteur 1 est ",
  prob1, "\n"
)
cat(
  "La probabilit√© pour l'effet du Facteur 2 est ",
  prob2, "\n"
)
